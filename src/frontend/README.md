# Frontend

Thư mục chứa giao diện người dùng của hệ thống quản lý luật giao thông.

## Cấu trúc thư mục

- `components/` - Các component tái sử dụng
- `pages/` - Các trang chính của ứng dụng
- `assets/` - Tài sản tĩnh (hình ảnh, CSS, fonts)
- `services/` - Các service API
- `utils/` - Các hàm tiện ích
